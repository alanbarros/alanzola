Sobre

Atualmente faço parte de uma equipe de desenvolvimento auto gerenciável que traz solução de software para a gestão ambiental e geográficas na Secretaria de Infraestrutura e Meio Ambiente do Estado de São Paulo.
Utilizamos o serviço de integração contínua da Microsoft (Azure DevOps) mesclado com metodologias ágeis como o Scrum para entregar nossos projetos com maior produtividade e qualidade.
As tecnologias que utilizo no dia a dia são:

  • C#
  • Microsoft .NET Framework (WebForms e MVC)
  • Microsoft SQL Server (SqlGeography usado com muita frequencia)
  • API Google Maps

Tenho experiências anteriores na área de infraestrutura que, unidas com minhas formações técnicas e de gestão me garantem uma ótima base para enfrentar novos desafios na área de desenvolvimento.
